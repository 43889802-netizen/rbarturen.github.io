Fórmulas de productos notables

Las matemáticas son fundamentales en nuestra vida diaria, ya que nos ayudan a resolver problemas, tomar decisiones financieras, medir cantidades, calcular distancias y comprender mejor el mundo que nos rodea. Desde administrar nuestro dinero hasta planificar actividades, las matemáticas están presentes en casi todas nuestras acciones cotidianas.

  1. (ax + b)(cx + d) = acx2 + (ad + bc)x + bd
  2. (a - b)(a2 + ab + b2) = a3 - b3
  3. (a + b)2 = a2 + 2ab + b2
  4. (a + b)(a - b) = a2 - b2
  5. (a - b)3 = a3 - 3a2b + 3ab2 - b3
  6. (x + a)(x + b) = x2 + (a + b)x + ab
  7. (a - b)2 = a2 - 2ab + b2
  8. (a + b)3 = a3 + 3a2b + 3ab2 + b3
  9. (a + b)(a2 - ab + b2) = a3 + b3
